Hexo-Next搭建个人博客（更换主题）

发表于 2019-09-16 更新于 2019-12-14 分类于 hexo 阅读次数： Valine：
本文字数： 3.4k字 阅读时长 ≈ 6 分钟

前置

已经搭建好的Hexo的平台，在本地可以用server打开的也算；

如果没有，请阅读Hexo-Next搭建个人博客（安装与部署）

主题

hexo主题，这里有所有的主题，任君选择，找好主题之后，从 Github 上面下载下来，本人使用的是NexT

安装

最简单的安装方法是克隆整个存储库：

或者，如果您需要任何其他变体，可以查看详细的安装说明。

应用

修改站点配置文件_config.yml把theme后改为next

配置

站点配置文件路径
~/hexo/_config.yml
主题配置文件路径
~/hexo/themes/next/_config.yml

站点

配置hexo网站相关信息

找到site的设置 按下面备注修改相关信息

设置hexo永久链接

设置永久链接的目的是为了便于分享和seo优化.这样别人收藏地址后不会出现链接失效

默认链接样式可修改为 (permalink配置只能有一种,关闭某种开头加 #)

主题配置

配置hexo中的tag,categories菜单

默认菜单只开启了首页和归档,hexo所有图标均来自fontawesome

手动生成hexo菜单对应文件 新菜单开启后是没有对应文件的,所以要手动生成about,tags,categories页面

新建about页面

新建一个tags页面

找到tags文件
hexo/source/tags/index.md编辑它

新建一个categories页面

找到categories文件
hexo/source/categories/index.md编辑它

配置hexo站点的footer信息

底部footer可以开关显示hexo版权,theme版权,还有建站时间等个性化配置

hexo网站next主题侧边栏宽度,头像信息,头像设置圆形和头像旋转
